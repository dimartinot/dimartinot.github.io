Moon's rock segmentation using Artificial Lunar Landscape Dataset¶

This project aims to show that artificialy generated data can be very useful to train Neural Networks for real-life tasks such as Rock Segmentation and detection.

The dataset used for this project was created by Romain PESSIA in an attempt to compensate the lack of such data in the field of space robotics.

I personally thank him for setting this data public.

Data Visualization¶

We will first introduce the data and try, then, to come up with a model that is able to predict rocks from a real Lunar picture.

There are two important folders of data:

  • "clean" that contains the cleaned ground-truth of the rocks, ground and sky positions
  • "render" that contains the input images

Let's load them in a pandas dataframe.

# Segmentation models losses can be combined together by '+' and scaled by integer or float factor
# set class weights for dice_loss (car: 1.; pedestrian: 2.; background: 0.5;)
dice_loss = sm.losses.DiceLoss(class_weights=np.array([0.5, 2, 1, 0.5]))
focal_loss = sm.losses.BinaryFocalLoss() if NUM_CLASS == 1 else sm.losses.CategoricalFocalLoss()
total_loss = dice_loss + (1 * focal_loss)

# actulally total_loss can be imported directly from library, above example just show you how to manipulate with losses
